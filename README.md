## 📖 Deskripsi

**Lembang Arena** adalah sistem booking lapangan futsal berbasis web yang memudahkan pengguna untuk memesan lapangan secara online. Aplikasi ini dilengkapi dengan dashboard admin untuk mengelola lapangan, booking, dan pengguna.

## ✨ Fitur Utama

### 👤 User
- Lihat daftar lapangan tersedia
- Booking lapangan dengan pilihan tanggal & waktu
- Riwayat booking
- Kalender booking

### 👨‍💼 Admin
- Dashboard dengan statistik & grafik
- Manajemen pengguna (CRUD)
- Manajemen lapangan (CRUD)
- Manajemen booking (CRUD)
- Kalender booking
